# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly). It uses bcrypt to hash the passwords, and cooke-session to encrypt cookies. 

## Features
- Create a user in session
- Login as a user (within the existing database)
- Add, delete, modify urls

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Styling
- Bootstrap

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
